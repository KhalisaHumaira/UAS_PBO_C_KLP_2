# UAS_PBO_C_KLP_2
Sistem MiniMarket 
Sistem Minimarket (Java CLI)

Ini adalah repository Tugas Besar Praktikum PBO. Aplikasi ini merupakan simulasi sistem kasir dan inventaris minimarket sederhana yang berbasis teks (Console/Terminal).

**Deskripsi Singkat Sistem**

Sistem ini dirancang dengan memisahkan dua jenis pengguna, yaitu:
1. Admin: Bertugas mengelola stok barang di toko. Admin bisa menambah barang baru, mengedit harga atau stok, menghapus barang, mencari barang, dan melihat laporan total pendapatan toko.
2. Customer: Bertindak sebagai pembeli. Customer bisa melihat daftar barang yang dijual, memasukkan barang ke keranjang belanja, dan melakukan checkout untuk mendapatkan struk pembayaran.
Sistem ini dibuat menggunakan bahasa pemrograman Java dan menerapkan konsep OOP (Object-Oriented Programming) seperti Encapsulation, Inheritance, Polymorphism, dan Abstraction.

**Cara Menjalankan Program**

Untuk menjalankan aplikasi ini ikuti langkah-langkah berikut:
1. Pastikan Java (JDK) sudah terinstall di laptop.
2. Download atau Clone repository ini.
3. Buka Terminal (jika di Mac/Linux) atau CMD (jika di Windows) dan arahkan ke dalam folder utama proyek ini.
4. Untuk meng-compile (menyiapkan) program, ketik perintah berikut di terminal lalu tekan Enter: cd minimarket javac Main.java
5. Setelah berhasil di-compile, jalankan program dengan mengetik perintah berikut lalu tekan Enter: java minimarket.Main
